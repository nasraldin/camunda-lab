## Global Constraints

- Fully deterministic; no AI/network
- `--fail-on error|warning` (default `error`)
- Exit `0` clean, `1` findings at/above fail level, `2` tool error
- Rule IDs stable (`bpmn/no-gateway-default`, â€¦) for suppressions later
- Fixtures in `testdata/bpmn/lint/`
- Does not deploy or call cluster APIs

## Initial rule set (MVP)

| Rule ID                            | Severity | Description                                                         |
| ---------------------------------- | -------- | ------------------------------------------------------------------- |
| `bpmn/process-start-event`         | error    | Process has a start event                                           |
| `bpmn/disconnected-element`        | error    | Element not connected on sequence flows (except justified isolates) |
| `bpmn/exclusive-gateway-default`   | warning  | Exclusive gateway should declare default flow                       |
| `bpmn/exclusive-gateway-condition` | error    | Non-default outgoing flows need conditions                          |
| `bpmn/duplicate-message-name`      | error    | Duplicate message names in model                                    |
| `bpmn/service-task-retry`          | warning  | Service task missing retry extension (Camunda 8)                    |
| `bpmn/timer-reachable`             | warning  | Timer event reachable from start (basic graph walk)                 |
